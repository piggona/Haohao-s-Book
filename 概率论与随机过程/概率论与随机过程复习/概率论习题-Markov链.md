> $f _ { i j } = P \left\{ T _ { i j } < \infty | \xi _ { 0 } = i \right\} = \sum _ { n = 1 } ^ { \infty } P \left\{ T _ { i j } = n | \xi _ { 0 } = i \right\} = \sum _ { n = 1 } ^ { \infty } f _ { i j } ^ { ( n ) }$
>
> $\mu _ { i j } = E \left( T _ { i j } \right) = \sum _ { n = 1 } ^ { \infty } n f _ { i j } ^ { ( n ) } $
